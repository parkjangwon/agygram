# Media capture guide

Add README screenshots or short GIFs here when preparing a public launch post.

Recommended captures:

1. Telegram `/start` onboarding checklist
2. Telegram `/menu` button panel
3. Telegram `/auth` URL/code flow with all secrets redacted
4. Telegram `/update` changelog and apply button

Before committing media:

- Blur bot tokens, user IDs, private chat names, emails, OAuth codes, workspace paths, and repository secrets.
- Prefer narrow mobile screenshots so the Telegram-first workflow is obvious.
- Keep GIFs short and under a few megabytes.
